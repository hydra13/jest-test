# jest-test

This is simple calculator for trying to use [Jest (JavaScript Testing Framework)](https://jestjs.io/)

## Operations:
- Addition
- Subtraction 
- Multiplication 
- Division 

## For getting project:
```
git clone https://github.com/hydra13/jest-test.git
cd jest-test
npm install
```

## For running tests:
```
npm run test
```
